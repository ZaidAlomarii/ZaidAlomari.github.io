# Zaid Alomari — Portfolio Website

A personal portfolio website built with pure HTML, CSS, and JavaScript.

## 📁 File Structure

```
portfolio/
├── index.html   ← Main HTML file
├── style.css    ← All styles
├── script.js    ← Interactivity (scroll reveal, mobile nav, skill bars)
├── photo.jpg    ← Your profile photo
└── README.md    ← This file
```

## 🚀 How to Deploy on GitHub Pages

1. Create a new GitHub repository named `your-username.github.io`
   (e.g. `ZaidAlomarii.github.io`)

2. Upload all files from this folder into that repository.

3. Go to **Settings → Pages** in your repo.

4. Under **Source**, select `main` branch and `/ (root)` folder, then click **Save**.

5. Your site will be live at `https://ZaidAlomarii.github.io` within a few minutes! 🎉

## 📝 Customization

- **Photo**: Replace `photo.jpg` with your own image (keep the same filename).
- **Content**: Edit `index.html` to update text, projects, or links.
- **Colors**: Change CSS variables at the top of `style.css` to restyle everything.

## 🛠 Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (Intersection Observer API)
- Google Fonts (Space Grotesk + Syne)
